Configure seu PIN Administrativo

Para proteger as configurações das rotinas, você precisa criar um PIN de 4 dígitos. Este PIN será solicitado sempre que você quiser acessar o painel de controle.

⚠️ Importante:

  • Use um PIN que você consiga lembrar
  • Não compartilhe este PIN com as crianças
  • O PIN é diferente da sua senha de login
  • Use apenas números (0-9)
